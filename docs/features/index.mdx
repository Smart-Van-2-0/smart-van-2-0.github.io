## Smart Van Specifications

Since this is an open source project, everyone is able to create their own
Smart Van Box and adapt it to their needs. For example, increasing the number
of controllable services, or integrating a different solar charger. Directly on
this site **you can find different Smart Van Boxes models to build yourself**
or to take example from.

**For each feature, its hardware requirements and related configurations are
specified**. In some cases there are several options to choose from, such as the
solar chargers required for energy monitoring. Or, if it will be necessary to
make configuration changes, for example by increasing the number of managed
services.

For more technical details on the features provided by Smart Van Boxes and
required by mobile apps, see the **Smart Van project <MapSpecs />**.
