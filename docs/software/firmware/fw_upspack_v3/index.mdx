Simple Python module that read data from [UPS Pack_V3](https://github.com/rcdrones/UPSPACK_V3/blob/master/README_en.md)
device and share them on the local DBus.<br />
This repository is part of the [Smart Van Project](https://www.smartvanbox.org/).

Once ran, this script **reads data from the serial specified port then notify
the DBus with updated values**. The DBus service and his properties depends on
the version get from the device (`SmartUPS` field). More info on [Supported
devices and value mapping](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.upspack_v3/#supported-devices-and-value-mapping).
