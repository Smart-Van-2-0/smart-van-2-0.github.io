Simple Python module that read data from [Sense Hat](https://www.waveshare.com/sense-hat-c.htm)
and share them on the local DBus.<br />
This repository is part of the [Smart Van Project](https://smartvanbox.org/).

Once ran, this script **reads data from the Sense Hat via I2C then notify
the DBus with updated values**. The DBus service and his properties are defined
into [_dbus_descs.py](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.sensehat/blob/main/fw_sensehat/sense/_dbus_descs.py) file.
More info on [Supported devices and value mapping](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.ioexp/#supported-devices-and-value-mapping).
