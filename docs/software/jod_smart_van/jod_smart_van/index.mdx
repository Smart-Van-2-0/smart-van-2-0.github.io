The core software driving the intelligence of the SmartVan.org project,
specifically embedded within the Smart Van Box. As the project's principal
software, the object agent defines the Smart Van Box's specifications, dynamically
adjusting capabilities based on its configuration.

This versatile agent incorporates additional software components, known as
firmwares, facilitating seamless communication with the underlying hardware.
Leveraging the [JODDistTMPL](https://github.com/johnosproject/com.robypomper.josp.jod.template)
project architecture, the IoT Agent is built upon the robust foundation of
the [John Object Daemon](https://github.com/johnosproject/com.robypomper.josp.jod)
from the John OS Project platform.

With the IoT Agent at the helm, the Smart Van Box becomes a sophisticated and
adaptive hub, delivering tailored functionalities according to its configuration.

Actually, the first version contains only the support to the solar charger
(e.g. the Victron SmartSolar or BlueSolar via Serial2USB). Checkout the
[FW Victron](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.victron/)
repository for more info on supported devices and which data are provided.

## System requirements

To run the JOD Smart Van distribution successfully, it's essential to have the
following software installed: Java and Python.

* Java 8 or higher: for JOD execution
* python3: for firmware execution
* python3-venv: for venv option on firmware execution

Additionally, depending on the enabled firmwares, it may be necessary to fulfill
their requirements. Here is the list of firmwares included in the JOD Smart Van
distribution:

* [FW Victron](/docs/software/firmware/fw_victron): v 1.0.2 [Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.victron/)
* [FW SIM 7600](/docs/software/firmware/fw_sim7600): v 1.0.0 [Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.sim7600/)
* [FW UPS Pack_V3](/docs/software/firmware/fw_upspack_v3): v 1.0.0 [Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.upspack_v3/)
* [FW Sense Hat](/docs/software/firmware/fw_sensehat): v 1.0.0 [Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.sensehat/)
* [FW IO Exp](/docs/software/firmware/fw_ioexp): v 1.0.0 [Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.ioexp/)

:::warning
Both the JOD Smart Van distribution and the firmwares are developed and tested
using linux. Therefore, it is recommended to use a linux system to run the JOD
Smart Van distribution.
:::
