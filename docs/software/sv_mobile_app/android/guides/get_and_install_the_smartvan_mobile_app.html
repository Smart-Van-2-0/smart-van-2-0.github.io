Software - Smart Van for Android

Get and Install the SmartVan Mobile App​

  1. Download the App
    From your smartphone, navigate to the SmartVan Mobile App page then to the Releases page; or directly to the downloads page.
    Download the latest version of the *.apk file from your smart phone.

  2. Install the App
    Proceed with the installation by opening the downloaded *.apk file.
    Your operating system may warn you about the potential risks of installing applications from third-party sources. Accept to continue with the installation.

  3. Launch the App
    Once installed, launch the SmartVan Mobile App.

If the SmartVanBox does not appear after launching the mobile app, ensure your phone is connected to the same WiFi network as the SmartVanBox.
