Welcome to the Smart Van official documentation.

Explore the Smart Van project on these pages. Start with the list of features
to see **how Smart Van can help you on your road adventures**. Or, for more
technical users, browse the project specifications to understand **how
information is exchanged** between the Smart Van Box and the Mobile App.

The Software and Hardware sections provide **all the details you need to update,
modify, or create your own Smart Van Box**. The Hardware section includes
documentation for different models of Smart Van Box, so you can choose the
option that best suits your needs. On the other hands, the Software section
includes not only the firmware references, but also the documentation for the
Smart Van Mobile App, including download links, user manuals, and source code.
