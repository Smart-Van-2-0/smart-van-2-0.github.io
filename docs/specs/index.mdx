Each Smart Van Box has its own characteristics such as integrated services,
connected sensors, buttons, etc. Each of these characteristics allows the Smart
Van Box to provide a set of <MapFeatures />. On the other hand, the mobile app
knows the information and actions provided by the Smart Box to which it is
connected and can therefore use them to implement its features.

The **meeting point between Smart Van Box and Features** are precisely these
characteristics defined in the specifications of the Smart Van project. In other
words, Smart Van Boxes provide these specifications and mobile applications use
them.
