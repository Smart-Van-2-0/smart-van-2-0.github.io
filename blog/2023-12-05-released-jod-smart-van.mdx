We are delighted to announce the release of JOD Smart Van version 1.0.0,
bringing a host of new features, improvements, and firmware updates. Here are
the key highlights of this release:

<img src="/img/software/jod_smart_van/jod_smart_van_logo_500.png" alt="JOD Smart Van logo" width="300"/>

### Major Updates
* **Build Configurations:** Implemented robust production and development build
  configurations to enhance project setup.
* **Firmware Management Scripts:** Integrated firmware management into
  `pre-startup.sh` and `post-startup.sh` scripts for improved control and
  efficiency.
* **Firmware Version Updates:**
  * [FW Victron](https://smartvanbox.org/docs/software/firmware/fw_victron) updated to v1.0.2. [View Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.victron/)
  * [FW SIM 7600](https://smartvanbox.org/docs/software/firmware/fw_sim7600) introduced with version 1.0.0. [View Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.sim7600/)
  * [FW UPS Pack_V3](https://smartvanbox.org/docs/software/firmware/fw_upspack_v3) added with version 1.0.0. [View Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.upspack_v3/)
  * [FW Sense Hat](https://smartvanbox.org/docs/software/firmware/fw_sensehat) included with version 1.0.0. [View Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.sensehat/)
  * [FW IO Exp](https://smartvanbox.org/docs/software/firmware/fw_ioexp) introduced with version 1.0.0. [View Sources](https://github.com/Smart-Van-2-0/com.robypomper.smartvan.fw.ioexp/)
* **File Structure Updates:**
  * Updated the struct.jod file to match Smart Van Specs.
  * Added structure files for all SV specs sub-groups.
  * Added structure files for any included firmware.
